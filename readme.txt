An easier way to link to Wikipedia.

== Description ==

WikiLink Wordpress plugin is an easier way to link to Wikipedia. Allows you to see a Wikipedia definition but without leaving your blog.

== Installation ==

1. Upload wikilink directory to the /wp-content/plugins/ directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure and enjoy :)

Once the plugin is activated it creates a "WikiLink" panel under "Options". It will allow you to manage the WikiLink options and styles.
